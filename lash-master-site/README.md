# Сайт-визитка мастера по наращиванию ресниц с админ-панелью

Полнофункциональный сайт для мастера по наращиванию ресниц с интерактивным расписанием и админ-панелью.

## 🌟 Функционал

### Публичная часть сайта:
- Главная страница с информацией о мастере
- Услуги и цены
- Портфолио работ
- Отзывы клиентов
- Форма записи
- **Интерактивное расписание на месяц** (зеленый - свободно, красный - занято)

### Админ-панель:
- Защищена логином и паролем
- Интерактивный календарь для управления записями
- Добавление/удаление записей клиентов
- Управление базой клиентов
- Просмотр расписания по месяцам

## 🚀 Развертывание

### 1. Установка Docker на Ubuntu:
```bash
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER
```

### 2. Клонирование проекта и запуск:
```bash
cd /path/to/lash-master-site
docker-compose up -d --build
```

### 3. Инициализация базы данных:
Откройте в браузере: `http://ваш-сервер/init-db`

Это создаст таблицы и учетную запись администратора.

## 🔐 Доступ к админ-панели

- **URL:** `http://ваш-сервер/admin/login`
- **Логин:** `admin`
- **Пароль:** `admin123`

**Важно:** После первого входа смените пароль!

## 🔧 Компоненты

- **PostgreSQL** - база данных для хранения записей и клиентов
- **Flask** - бэкенд API и админ-панель
- **Nginx** - раздача статики и проксирование запросов к бэкенду

## 💾 Хранение данных

Данные базы данных сохраняются в Docker volume `postgres_data`, что обеспечивает сохранность данных при перезапуске контейнеров.

## 🔒 Безопасность

Измените секретный ключ в `docker-compose.yml`:
```yaml
SECRET_KEY: ваш-уникальный-секретный-ключ
```

И смените пароль администратора после первого входа через админ-панель.

## 🛠 Управление

```bash
# Просмотр логов
docker-compose logs -f

# Остановка
docker-compose down

# Перезапуск
docker-compose restart

# Пересборка при изменениях
docker-compose up -d --build
```

## 📞 Поддержка

При возникновении проблем проверьте логи:
```bash

```
